## ✨ Features

- **Double-Theme System:**
  - **Ghost (Dark):** A premium deep cardinal-red and obsidian theme for a professional "Dark-Mode" first look.
  - **Zen (Light):** A dramatic "Blood-Red" and Ice-White combination for high contrast and unique visual identity.
- **Advanced Bento Grid Layout:** A responsive Hero section with magnetic elements, floating UI badges, and tech-radar cards.
- **Adaptive Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop with linearized timelines for small screens.
- **High Performance:** Optimized for Core Web Vitals with Vite, rollup-chunk splitting, and efficient asset management.
- **Interactive Micro-animations:** Powered by Framer Motion, including custom cursors (on desktop), hover rays, and stagger reveals.
- **Rich Contact System:** Direct contact links with integrated UI/UX feedback.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Animations:** Framer Motion
- **Styling:** Vanilla CSS3 (Custom Variables/Tokens), Glassmorphism
- **Icons:** Lucide-React
- **Architecture:** Component-based modular architecture

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/daief48/portfolio-daief-2026.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start development server:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```
